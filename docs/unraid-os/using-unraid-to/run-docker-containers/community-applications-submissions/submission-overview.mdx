# Submitting Community Applications

Community Applications submissions are handled through the [Community Apps submission portal](https://ca.unraid.net/submit). Use the portal to connect your Unraid account, add a public GitHub repository, validate and scan the repository, fix any reported issues, and submit it for review.

## What you submit

Community Applications accepts a public, active GitHub repository that contains Unraid Docker template XML files, plugin wrapper XML files, or both. The repository must also include repository-level metadata in `ca_profile.xml`.

Start from the [Community Apps starter repository](https://github.com/unraid/unraid-community-apps-starter) whenever possible. It provides the expected file layout and examples that are designed to work with the submission scanner.

## Requirements

Before submitting, make sure your repository:

- Is public, active, and not archived or disabled.
- Includes an OSI-approved open source license for the repository contents.
- Includes valid Docker template XML files and/or plugin wrapper XML files.
- Includes `ca_profile.xml` at the repository root with a non-empty `<Profile>` section.
- Replaces every placeholder URL, name, icon, and support link from any starter files.
- Has a clear support path, such as an Unraid forum thread, issue tracker, documentation site, or Discord community.

Container image licenses are separate from the license used for templates, metadata, and documentation in the submission repository.

## Submission flow

1. Prepare the GitHub repository and confirm all required files are committed.
2. Sign in to the [submission portal](https://ca.unraid.net/submit) with your Unraid account.
3. Add the GitHub repository URL.
4. Run validation and scan checks in the portal.
5. Fix reported issues in the repository, then rerun validation and scan.
6. Submit the repository for Community Applications moderation review once checks pass.

Use validation and scan after each meaningful XML change. This keeps scanner feedback focused and makes review easier.
