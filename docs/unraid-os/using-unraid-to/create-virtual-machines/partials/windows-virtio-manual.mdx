1. Download the [latest stable VirtIO drivers ISO](https://fedorapeople.org/groups/virt/virtio-win/direct-downloads/stable-virtio/virtio-win.iso) and save it to your Unraid `isos` share.

   :::tip[Verify download integrity]

   Download the [CHECKSUM file](https://fedorapeople.org/groups/virt/virtio-win/direct-downloads/stable-virtio/CHECKSUM) from the same directory to verify the ISO's integrity. This helps ensure your download is complete and hasn't been tampered with.

   :::

2. Edit your %%VM|vm%% settings to point to the new ISO in the `isos` share.
3. In Windows Device Manager:
   - Right-click devices with yellow warnings
   - Select **Update driver â†’ Browse my computer**
   - Navigate to the %%VirtIO|virtio%% ISO drive (usually `D:\` or `E:\`)
4. Reboot after updating all drivers.
