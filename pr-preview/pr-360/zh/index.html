Unraid 文档

欢迎使用官方的 Unraid 文档！这一全面的资源旨在帮助您充分利用您的 Unraid 服务器，无论您是在设置首个系统，还是管理一个复杂的家庭实验室环境。

开始入门​

如果您是 Unraid 新手，建议先阅读这些基本指南：

  • Unraid 是什么？ - 了解 Unraid 的核心功能和优势
  • 设置 Unraid - 从安装到首次启动的完整设置指南
  • 浏览 WebGUI - 自信地导航 WebGUI

需要帮助吗？​

社区论坛​

Unraid论坛是Unraid社区的核心。在这里你可以找到：

  • 用户讨论和故障排除
  • 插件和 Docker 容器支持
  • 硬件推荐和兼容性讨论
  • 社区开发的解决方案和变通方法

官方资源​

  • Unraid 首页 - 产品信息、下载和许可
  • 社区应用程序 - 浏览和安装插件及 Docker 容器
  • Unraid Connect - 远程管理仪表板

支持选项​

  • 社区支持：在论坛中发布问题
  • 文档支持：使用搜索功能和导航来查找答案
  • 插件支持：检查社区应用程序部分中的单个插件主题

术语表术语​

在本文档中，您会注意到以 WebGUI 格式的术语。这些是术语表中的术语，提供快速定义和解释。

要使用它们，将鼠标悬停在术语表术语上以查看包含定义的工具提示。 您可以在 术语表页面 中查看整个术语表。

贡献​

发现错误或有建议？该文档是开源的，并托管在 GitHub 。我们欢迎来自社区的贡献！

准备好深入了解了吗？ 从 入门指南 部分开始，或者使用搜索功能查找特定信息。如果您找不到所需内容，Unraid论坛 始终为您提供帮助！
