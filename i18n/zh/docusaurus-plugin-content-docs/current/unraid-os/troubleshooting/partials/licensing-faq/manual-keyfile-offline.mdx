1. 请确保您的 USB 驱动器有最新备份。请使用 [Unraid Connect](../../../../unraid-connect/overview-and-setup.mdx)（推荐），或使用位于 ***Main → Boot Device → Boot Device Backup*** 的本地备份选项（在 7.3.0 之前的版本中，请查看 **Flash** / **Flash device**）。
2. 关闭您的 Unraid 服务器并移除 USB 启动设备。
3. 将 USB 驱动器插入另一台计算机。
4. 打开 USB 驱动器，将您的 `.key` 文件复制到 `/config` 文件夹中。
   *确保这是唯一的 `.key` 文件——删除其他文件。*
5. 安全弹出 USB 驱动器，将其重新安装到您的服务器中，然后重启。
