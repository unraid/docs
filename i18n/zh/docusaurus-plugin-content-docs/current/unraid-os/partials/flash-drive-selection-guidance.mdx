Unraid 要求启动介质具有唯一的硬件 %%GUID|guid%%。对于 USB 启动，这通常是一个 USB 闪存盘。缺少唯一 %%GUID|guid%%，或重复使用已注册给其他用户的 %%GUID|guid%% 的驱动器，不能与 Unraid 一起使用；如果它们造成注册问题，可能会被列入黑名单。

最广为人知的品牌，如 SanDisk、Kingston 和 Samsung，经常被伪造。假冒驱动器通常没有唯一的 %%GUID|guid%%，而且据报告，这些品牌的一些正品型号也使用非唯一标识符。如果你不确定自己是否从授权零售商处购买了正品，选择一个不那么流行的品牌实际上可能更安全。这些品牌通常提供高质量产品，但不太容易成为造假者的目标。

社区中通常认为因具有唯一 %%GUIDs|guid%%（并且一般较不易被仿冒）而值得信赖的品牌包括：

- TeamGroup
- Transcend
- PNY

请从**信誉良好**的零售商处购买，并避免拍卖网站、未知卖家以及二手或灰色市场驱动器。有关 SanDisk 的更多信息，请参阅[论坛上关于假冒 SanDisk USB 驱动器的 PSA](https://forums.unraid.net/topic/119052-psa-on-sandisk-usbs/)。
