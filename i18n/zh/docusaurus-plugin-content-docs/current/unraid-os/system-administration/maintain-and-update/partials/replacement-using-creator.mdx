*推荐大多数用户使用*

1. 下载适用于Windows或macOS的[Unraid USB闪存创建工具](https://unraid.net/download)。
2. 将新的USB闪存设备插入计算机。
3. 在创建工具中，点击操作系统，向下滚动至**使用自定义**并浏览到您的备份ZIP文件。

<div style={{ margin: 'auto', maxWidth: '600px', display: 'flex', flexDirection: 'column', alignItems: 'center' }}>
  ![选择版本](/img/Selectversion.png)
</div>

4. 选择新的USB闪存设备作为目标并单击**写入**。
5. 关闭服务器，用新设备替换旧闪存设备并打开电源。
6. 当提示`无效、缺少或过期的注册码`时，选择**注册码**。

<div style={{ margin: 'auto', maxWidth: '600px', display: 'flex', flexDirection: 'column', alignItems: 'center' }}>
  ![无效密钥](/img/Invalidkey.png)
</div>

7. 如有需要，将现有的许可证密钥文件复制到新设备的`boot/config`中。
8. 确保config文件夹中仅存在一个密钥文件。
9. 在***工具→注册码***中选择**更换密钥**并输入您的电子邮件地址。

<div style={{ margin: 'auto', maxWidth: '600px', display: 'flex', flexDirection: 'column', alignItems: 'center' }}>
  ![无效密钥](/img/Replacekey.png)
</div>

10. 按照电子邮件中的说明安装您的新密钥。
11. 完成后，选择**完成**。

:::important

如果您看到"密钥文件无效"错误，您的密钥可能被列入黑名单，或不是最后一个有效密钥。 [联系 Unraid 支持](https://unraid.net/support)以获取帮助。

:::
