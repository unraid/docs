*如果USB闪存创建工具不适用于您的系统，请使用此方法*

:::important[Before 开始]

Use the manual method only if you are comfortable with basic file management and your system does not support the USB flash creator.

:::

1. Prepare a new flash device following the [Manual install method](../../../getting-started/set-up-unraid/create-your-bootable-media.mdx#manual-install-method).
2. Copy your backed-up `config` folder to the new flash device, overwriting existing files.
3. Shut down the server, swap the flash devices, and power on.
4. 当提示`无效、缺少或过期的注册码`时，选择**注册码**。

<div style={{ margin: 'auto', maxWidth: '600px', display: 'flex', flexDirection: 'column', alignItems: 'center' }}>
  ![无效密钥](/img/Invalidkey.png)
</div>

5. 在***工具→注册码***中选择**更换密钥**并输入您的电子邮件地址。

<div style={{ margin: 'auto', maxWidth: '600px', display: 'flex', flexDirection: 'column', alignItems: 'center' }}>
  ![无效密钥](/img/Replacekey.png)
</div>

6. 按照电子邮件中的说明安装您的新密钥。
7. 选择 **完成**。
