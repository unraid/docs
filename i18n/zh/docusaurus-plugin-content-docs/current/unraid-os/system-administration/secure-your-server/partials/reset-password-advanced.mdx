这种方法仅移除 **%%root|root-user%% 密码**，其他用户不受影响。

通过此方法重置：

1. **关闭** 您的 Unraid 服务器。

2. **连接 USB 闪存驱动器** 到计算机（Windows 或 Mac）。

3. **使用文本编辑器**（如 Notepad++）打开位于 `/config/shadow` 的文件。

4. 编辑如下所示的行：
   `root:$&$&%*1112233484847648DHD$%.:15477:0:99999:7:::`

   将其更改为：

   `root::15477:0:99999:7:::`

5. **保存更改** 到文件并将 USB 驱动器重新连接到您的服务器。

6. **启动您的 Unraid 服务器** 并设置新的 root 密码。
