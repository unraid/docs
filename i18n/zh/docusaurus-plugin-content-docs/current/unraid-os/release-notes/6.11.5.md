## 版本 6.11.5 2022-11-20

此版本恢复了一个错误更改了文件的更改。这导致无法选择“macvlan”自定义docker网络类型。现在仅新安装将默认选择“ipvlan”。

### Docker

- 修复：仅对新安装将 IPVLAN 设置为默认。
