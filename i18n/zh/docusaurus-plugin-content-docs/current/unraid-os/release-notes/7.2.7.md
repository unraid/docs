# 发行说明 · Unraid OS 7.2.7

## 版本 7.2.7 2026-05-21

本次发布重点关注安全更新、Docker 更新和 WebGUI 修复，并对核心系统组件的软件包进行了刷新。

对于对安全敏感的环境，建议升级。

## 升级

如需分步说明，请参阅[更新 Unraid](/unraid-os/updating-unraid/)。有关您的[许可证](/unraid-os/troubleshooting/licensing-faq/)的问题？

## 相较于 [7.2.6](/unraid-os/release-notes/7.2.6/) 的变更

### 安全

- Docker、nginx、glibc、jq、avahi 以及 glibc 支持库的安全更新。

### 容器 / Docker

- Docker 29.5.1
- Docker VLAN 容器在更新后会保留其已配置的网关。
- 在包含大量容器的页面上，Docker 容器菜单仍可访问。

### WebGUI / 系统

- 服务器插件中已恢复 Connect 版本检查。

### Linux kernel

- 内核升级：6.12.90-Unraid（CVE-2026-43490、CVE-2026-46333）
- 影响启用 Docker 的系统以及部分 USB 3.1 设备的性能回归问题。
