# 版本 7.2.2 2025-11-21

如果您已经在版本 7.2.1 上，这次更新是一个 **可选** 更新。此更新主要是针对试用用户，
修复了一个导致他们无法看到试用期到期日期的错误。同时，我们还解决了与自定义背景标题颜色相关的问题，
如果您设置了这些颜色，建议您更新。

如果您在版本 7.2.0 或更早版本上，您可以直接跳到 7.2.2。

## 升级

有关分步说明，请参阅[升级 Unraid](../../unraid-os/system-administration/maintain-and-update/upgrading-unraid.mdx)。
