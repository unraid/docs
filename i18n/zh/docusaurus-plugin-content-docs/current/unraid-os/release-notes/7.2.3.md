## 与[7.2.2](7.2.2.md)的变更对比

### 存储

- 修复：升级了 Samba 以解决 Time Machine 的问题
- 修复：仅在重启后检测到预清盘磁盘签名

### webGUI

- 修复：解决了在标题背景图像上应用渐变方式的回归问题
- 修复：通知文本有时显示颜色不正确，难以阅读
- 修正：在_**设置 → 通知设置 → SMTP设置**_中测试SMTP配置时的潜在XSS问题

### 其他

- 修正：DNS设置可能被网络中的其他设备覆盖（禁用avahidnsconfd）
- 修正：Docker XML模板中的Readme未填充
- 修正：缺少novnc文件导致系统日志中出现警告

### Unraid API

- dynamix.unraid.net 4.28.2 - [查看更改](https://github.com/unraid/api/releases)

### Linux kernel

- 版本6.12.54-Unraid (无变化)

### 基础发行版更新

- samba: 版本 4.23.4
