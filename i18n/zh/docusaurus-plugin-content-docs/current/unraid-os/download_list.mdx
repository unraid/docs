# 版本存档

使用此存档查找 Unraid OS 版本的发布说明和下载链接。要更新现有服务器，请使用内置更新程序；有关步骤，请参阅[标准更新流程](./updating-unraid/index.mdx#standard-update-process)。要创建新的启动设备，请使用 [Unraid USB Creator](https://unraid.net/download)。

存档按支持状态对版本进行分组。在受支持的次版本系列中，只有最新的补丁版或预发布版本会在适当情况下收到更新；同一系列中的较旧版本会被取代。标记为 **EOL** 的版本不受支持，也不再包含更新。

如果你正在决定要安装哪个版本，请先查看 [Unraid OS 版本类型](./updating-unraid/release-types.mdx)。有关许可证资格的详细信息，请参阅 [许可常见问题](./troubleshooting/licensing-faq.mdx#no-extension)。
