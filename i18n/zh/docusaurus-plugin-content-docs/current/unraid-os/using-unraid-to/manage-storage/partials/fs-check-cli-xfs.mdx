通过命令行检查 %%XFS|xfs%% 文件系统：

- 在**维护模式**下启动 %%array|array%%。
- 运行以下命令：`xfs_repair -v /dev/mdXp1`
- 将 `X` 替换为磁盘编号（例如 `/dev/md1p1`）。
- 对于加密的 %%XFS|xfs%%，使用 `/dev/mapper/mdXp1`。
- 对于不在 %%array|array%% 的驱动器： `xfs_repair -v /dev/sdX1`
- 确保您使用了正确的设备标识符。

:::warning
在不处于**维护模式**的 %%array|array%% 磁盘上运行此命令将使 %%parity|parity%% 无效。
:::
