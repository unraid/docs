:::info[Common 用例]

在维护后或添加新的缓存设备以提高性能时将文件移回%%cache pool|cache-pool%%。

:::

要从 %%array|array%% 移动文件到一个存储池：

1. 进入 **设置** 并关闭 Docker 和 %%VM|vm%% 管理器，以防止任何打开的文件干扰。
2. 在 **共享** 选项卡中，为每个要移动的共享（例如 `appdata` 或 `system`），将 **主存储** 设置为目标 %%cache pool|cache-pool%% 并将 **次存储** 设置为 %%array|array%%。将 **搬家操作** 设置为 **阵列 → 缓存**。
3. 进入 **主页面** 选项卡并点击 **立即移动**，以开始将文件从 %%array|array%% 移动到 %%cache pool|cache-pool%%。
4. %%Mover|mover%% 完成后，检查文件是否已在 %%cache pool|cache-pool%% 上。
5. 移动完成后，在 **设置** 中重新开启 Docker 和 %%VM|vm%% 管理器。
6. （可选）要设置共享仅使用 %%cache pool|cache-pool%%：如果您希望某个共享的所有文件都保持在 %%cache pool|cache-pool%% 上，将 **主存储** 设置为您的 %%cache pool|cache-pool%% ，并将 **次存储** 设置为 **无**。
