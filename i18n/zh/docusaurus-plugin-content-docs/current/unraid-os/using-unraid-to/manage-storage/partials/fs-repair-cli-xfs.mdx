- 在**维护模式**下启动 %%array|array%%。
- 运行以下命令：`xfs_repair /dev/mdXp1`。将`X`替换为磁盘编号（例如，`/dev/md1p1`）。
- 对于加密的 %%XFS|xfs%%，使用：`/dev/mapper/mdXp1`。
- 如果系统提示使用`-L`，请重新运行命令如下：`xfs_repair -L /dev/mdXp1`。通常，这是完成修复所需且安全的操作。
- 对于不属于%%array|array%%的驱动器：`xfs_repair /dev/sdX1`。

:::caution
在未进入 **%%Maintenance Mode|maintenance-mode%%** 的情况下，对 %%array|array%% 磁盘运行此命令会导致校验和无效。
:::
