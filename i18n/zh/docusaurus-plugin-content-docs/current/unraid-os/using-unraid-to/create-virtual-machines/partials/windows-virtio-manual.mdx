1. 下载 [最新稳定的 VirtIO 驱动程序 ISO](https://fedorapeople.org/groups/virt/virtio-win/direct-downloads/stable-virtio/virtio-win.iso) 并保存到您的 Unraid `isos` 共享中。

   :::提示[验证下载完整性]
   从同一目录下载 [检查文件](https://fedorapeople.org/groups/virt/virtio-win/direct-downloads/stable-virtio/CHECKSUM)来验证ISO的完整性。这有助于确保您的下载是完整的且未被篡改。
   :::

2. 编辑您的 %%VM|vm%% 设置以指向 `isos` 共享中的新 ISO。

3. 在 Windows 设备管理器中：
   - 右键单击带黄色警告的设备
   - 选择 **更新驱动程序 → 浏览我的计算机**
   - 导航至 %%VirtIO|virtio%% ISO 驱动器（通常为 `D:\` 或 `E:\`）

4. 更新所有驱动程序后重新启动。
