如果您需要重新安装 Docker 容器或插件（例如，如果您的 `docker.img` 文件已损坏），Community Applications 为您简化了这一过程。所有应用程序设置都作为 Docker 模板 XML 文件保存在 `/boot/config/plugins/dockerMan/templates-user` 中。有关 Docker 模板 XML 架构和自定义标签的更多信息，请参阅 Unraid 论坛上的 [Docker Template XML Schema documentation](https://forums.unraid.net/topic/38619-docker-template-xml-schema/)。

要重新安装应用程序：

1. 转到%%WebGUI|WebGUI%%的**应用**选项卡。
2. 选择**以前的应用**过滤器。
3. 找到您想要重新安装的应用程序并按照提示进行操作。

您的已保存设置将自动恢复，确保快速、简便的恢复。
