title: 激活码兑换
description: 如何兑换激活码以安装您的 Unraid 许可证
---

:::info\[Prerequisite]

A minimum of Unraid OS v6.10 or later, or access to your Unraid account dashboard.

:::

## 如何兑换您的激活码

首先，请遵循我们的[入门指南](../unraid-os/getting-started/what-is-unraid.mdx)来设置Unraid服务器。

当您的服务器已启动并运行时，请从该服务器开始，这样您的许可证就会应用到正确的服务器上。

### 对于 Unraid 6.10及更高版本

您可以从以下任一位置开始：

1. 登录到您的 Unraid 服务器的 WebGUI（默认情况下为 [http://tower](http://tower) 或 Mac 上的 [http://tower.local](http://tower.local)）。

2. 您可以使用以下任一入口点：

   - 导航到 **Tools > Registration**，然后在 **Actions** 中选择 **Redeem Activation Code**。

   ![显示 Redeem Activation Code 操作的 Tools Registration 页面](/img/unraid-account/tools-registration-ui.png)

   - 打开右上角的 Unraid.net 或 Connect 菜单并选择 **管理许可证**。

   ![显示“管理许可证”操作的 Unraid.net 帐户菜单](/img/unraid-account/upc-ui.png)

3. 如果您看到 **缺少密钥文件** 消息，请使用相同的入口：

   - 在 **工具 > 注册** 中，在 **操作** 下选择 **兑换激活码**。

   ![带有“缺少密钥文件”警告和“兑换激活码”操作的工具注册页面](/img/unraid-account/missing-key-tools-registration-ui.png)

   - 从右上角的 Connect 面板中选择 **管理许可证**。

   ![显示“缺少密钥文件”警告和“管理许可证”操作的 Connect 面板](/img/unraid-account/missing-key-upc-ui.png)

4. 登录到您的 Unraid.net 帐户，或者在提示时创建一个帐户。

5. 在帐户应用中，如果 **兑换激活码** 尚未被选中，请选择它。

   ![帐户应用概览，显示“兑换激活码”为可用操作](/img/unraid-account/account-ui-landing-page-no-code.png)

6. 输入您的激活码，或者如果显示了您帐户中的一个激活码，则选择它，然后点击 **兑换代码**。

   ![帐户应用“兑换激活码”页面，带有激活码输入框和“兑换代码”按钮](/img/unraid-account/redeem-code-account-ui.png)

7. 点击 **点击此处完成您的许可证安装（必需）**，以完成在服务器上安装许可证。

   ![用于完成许可证安装的帐户应用按钮](/img/unraid-account/finalize-install-button.png)

安装完成后，许可证将应用到您的服务器。

### 对于 Unraid 6.10及更高版本

升级激活码也使用相同的 **兑换激活码** 页面。帐户应用会显示或接受可用于您当前许可证的代码。

### 对于 Unraid 6.9 及更早版本

1. 在 %%WebGUI|web-gui%% 中导航到 **工具 > 注册** 页面。

2. 选择 **兑换激活码** 或 **购买密钥**。

3. 向下滚动找到“兑换升级激活码”输入框。

## 重要说明

- 激活码只能使用一次。
- 确保您使用的是购买许可证的相同帐户登录。
- 激活码必须可用于您要激活的服务器。
- 如果帐户应用未显示您的服务器或兑换表单，请从 %%WebGUI|web-gui%% 中的 **工具 > 注册** 或 **管理许可证** 重新打开它。

## 需要帮助吗？

如果您在兑换激活码时遇到问题：

- 确保您准确输入了电子邮件中的代码
- 检查激活码是否已使用
- 确保您使用的是购买许可证同一账户登录
- 如果帐户应用未显示正确的服务器，请从您的服务器再次打开它
- 有关更多信息，请查看我们的[许可常见问题](../unraid-os/troubleshooting/licensing-faq.mdx)。
