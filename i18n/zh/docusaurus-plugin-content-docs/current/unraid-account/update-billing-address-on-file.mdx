## 如何更新您存档的账单地址

您可以在下载发票 PDF 之前，更新发票上显示的账单地址。

1. 在订单确认页面上的 **发票账单地址** 部分中，点击 **更新地址**。
2. 如果您已经离开该页面，请登录您的 Unraid 账户，前往 **管理购买** -> **订单**，打开该订单，然后点击 **更新地址**。
3. 查看发票账单地址预览。
4. 如果地址正确，请点击 **立即下载发票**。
5. 如果需要更改地址，请点击 **更新地址** 以打开 Stripe 客户门户，更新您的账单地址并保存，然后再次下载发票。

![带有“更新地址”选项的订单详情页面](/img/unraid-account/update-billing-address-new-step-2.png)

![下载发票前的发票账单地址预览](/img/unraid-account/update-billing-address-new-step-3.png)

## 接下来会发生什么？

在 Stripe 中保存更改后，您下载的发票 PDF 将使用更新后的账单地址。

## 如果您需要帮助

- 请确保您已登录用于购买的同一账户。
- 如果您在 **管理购买项目** -> **订单** 中看不到该订单，请确认您使用的是正确的账户。
- 如果您仍然无法访问该订单或更新地址，请联系支持团队。
