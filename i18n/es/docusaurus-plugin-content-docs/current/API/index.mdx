# Bienvenido a la API de Unraid

A partir de Unraid OS v7.2, la API está integrada en el sistema operativo - ¡no se requiere instalación de plugins!

La API de Unraid proporciona una interfaz %%GraphQL|graphql%% para la interacción programática con tu servidor Unraid. Permite capacidades de automatización, supervisión e integración mediante una API moderna y fuertemente tipada, con múltiples métodos de autenticación (claves de API, cookies de sesión y %%SSO|sso%%/OIDC), cobertura completa del sistema y herramientas de desarrollo integradas.

## Disponibilidad

### Integración nativa (Unraid 7.2+)

A partir de Unraid 7.2, la API está integrada directamente en el sistema operativo:

- No se requiere instalación de plugins
- Disponible automáticamente al iniciar el sistema
- Integración profunda con el sistema
- Acceso a través de ***Settings → Management Access → API***

### Instalación del plugin (anteriores a 7.2 y usuarios avanzados)

Para versiones de Unraid anteriores a 7.2, o para acceder a funciones más nuevas de la API:

1. Instala el plugin [Unraid Connect](../unraid-connect/overview-and-setup.mdx) desde [Community Applications](../unraid-os/using-unraid-to/run-docker-containers/community-applications.mdx).
2. [Configure el plugin](./how-to-use-the-api.mdx#enabling-the-graphql-sandbox).
3. Accede a la funcionalidad de la API a través del %%GraphQL|graphql%% Sandbox.
