## Cambios frente a [7.2.2](7.2.2.md)

### Almacenamiento

- Arreglo: Upgrade de Samba para resolver problemas con Time Machine.
- Arreglo: La firma del disco preaclarada solo se detecta después de un reinicio.

### webGUI

- Arreglo: Regresión en cómo se aplican los degradados sobre imágenes de fondo en el encabezado.
- Arreglo: El texto de notificación a veces se muestra en colores incorrectos, lo que dificulta su lectura.
- Solución: Problema potencial de XSS al probar la configuración de SMTP en _**Configuración → Configuración de Notificaciones → Configuración de SMTP**_

### Varios

- Solución: La configuración de DNS podría ser sobrescrita por otros dispositivos en la red (deshabilitar avahidnsconfd)
- Solución: El archivo Readme no se estaba mostrando en las plantillas XML de Docker
- Solución: Archivos novnc faltantes que causaban advertencias en el syslog

### API de Unraid

- dynamix.unraid.net 4.28.2 - [ver cambios](https://github.com/unraid/api/releases)

### Núcleo de Linux

- versión 6.12.54-Unraid (sin cambios)
