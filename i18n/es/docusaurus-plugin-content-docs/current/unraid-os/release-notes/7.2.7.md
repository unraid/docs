# Notas de la versión · Unraid OS 7.2.7

## Versión 7.2.7 2026-05-21

Esta versión se centra en actualizaciones de seguridad, actualizaciones de Docker y correcciones de WebGUI, junto con la actualización de paquetes para los componentes principales del sistema.

Para entornos sensibles a la seguridad, se recomienda actualizar.

## Actualización

Para instrucciones paso a paso, consulta [Actualización de Unraid](/unraid-os/updating-unraid/). ¿Tienes preguntas sobre tu [licencia](/unraid-os/troubleshooting/licensing-faq/)?

## Cambios frente a [7.2.6](/unraid-os/release-notes/7.2.6/)

### Seguridad

- Actualizaciones de seguridad para docker, nginx, glibc, jq, avahi y las bibliotecas de soporte de glibc.

### Contenedores / Docker

- Docker 29.5.1
- Los contenedores VLAN de Docker conservan su puerta de enlace configurada después de las actualizaciones.
- Los menús de contenedores de Docker siguen siendo accesibles en páginas con muchos contenedores.

### WebGUI / Sistema

- Las comprobaciones de versión de Connect se han restaurado en el complemento del servidor.

### Núcleo de Linux

- actualización del kernel: 6.12.90-Unraid (CVE-2026-43490, CVE-2026-46333)
- Regresión de rendimiento que afecta a los sistemas con Docker y a algunos dispositivos USB 3.1.
