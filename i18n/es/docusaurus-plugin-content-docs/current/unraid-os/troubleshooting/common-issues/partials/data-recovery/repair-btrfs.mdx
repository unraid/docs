- Use `btrfs scrub` en el %%WebGUI|web-gui%% para escanear y reparar ciertos errores.
- En discos únicos, el scrub puede detectar pero no reparar algunos errores; si es así, considere copiar datos y reformatear el disco o grupo.

:::note

- Las herramientas de reparación %%BTRFS|btrfs%% actuales pueden no abordar toda la corrupción; consulte la documentación de Unraid para obtener herramientas actualizadas si es necesario.
- Para obtener más detalles sobre scrub y recuperación, consulte la documentación respectiva de %%BTRFS|btrfs%% o los foros de Unraid para obtener orientación:::
