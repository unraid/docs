# Archivo de versiones

Use este archivo para encontrar las notas de la versión y los enlaces de descarga de las versiones de Unraid OS. Para actualizar un servidor existente, use el actualizador integrado; consulte el [proceso de actualización estándar](/unraid-os/updating-unraid/#standard-update-process) para conocer los pasos. Para crear un nuevo dispositivo de arranque, use el [Unraid USB Creator](https://unraid.net/download).

El archivo agrupa las versiones por estado de soporte. Solo la compilación de parche o de prelanza más reciente de una serie menor compatible recibe actualizaciones según corresponda; las compilaciones anteriores de la misma serie quedan reemplazadas. Las versiones marcadas como **EOL** no tienen soporte y ya no incluyen actualizaciones.

Si estás decidiendo qué versión instalar, empieza por [tipos de versiones de Unraid OS](./updating-unraid/release-types.mdx). Para obtener detalles sobre la elegibilidad de la licencia, consulta las [Preguntas frecuentes sobre licencias](./troubleshooting/licensing-faq.mdx#no-extension).
