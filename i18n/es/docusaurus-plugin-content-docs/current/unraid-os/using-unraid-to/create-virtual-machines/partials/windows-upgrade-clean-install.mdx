1. Descarga el [ISO de Windows 11](https://www.microsoft.com/en-us/software-download/windows11).
2. Suba el ISO a su carpeta `isos` en su servidor Unraid.
3. Modifica tu plantilla del %%VM|vm%% para arrancar desde el ISO en la compartición `isos`.
4. Arranca el %%VM|vm%% desde el ISO y realiza la instalación limpia.
5. Reinstala tus aplicaciones y restaura tus datos desde la copia de seguridad.
