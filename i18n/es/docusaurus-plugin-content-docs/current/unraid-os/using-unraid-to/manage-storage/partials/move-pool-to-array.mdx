:::info[Common caso de uso]

Mover archivos del %%cache pool|cache-pool%% al %%array|array%% antes de realizar mantenimiento o actualizaciones para asegurar que tus datos estén seguros.

:::

Para mover archivos de tu agrupación a la %%array|array%%:

1. Vaya a **Configuraciones** y apague Docker y el Administrador de %%VM|vm%%. Esto evita que se mantengan archivos abiertos, permitiendo que el %%Mover|mover%% transfiera todo sin problemas.
2. In the **Shares** tab, for each share you want to move (like `appdata` or `system`), set **Primary storage** to the source %%cache pool|cache-pool%% and **Secondary storage** to the %%array|array%%. Set **Mover action** to **cache → array**.
3. Go to the **Main** tab and click **Move Now** to transfer files from the %%cache pool|cache-pool%% to the %%array|array%%.
4. Después de que el %%Mover|mover%% termine, verifique que los archivos se hayan movido haciendo clic en el icono de carpeta junto a la entrada del caché en la pestaña **Principal**.
5. Una vez que todos los archivos estén en el %%array|array%%, puede volver a encender con seguridad los servicios de Docker y %%VM|vm%%.
