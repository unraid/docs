Para verificar un sistema de archivos %%XFS|xfs%% a través de la línea de comandos:

- Inicie el %%array|array%% en **Modo de Mantenimiento**.
- Ejecute el siguiente comando: `xfs_repair -v /dev/mdXp1`
- Reemplace `X` con el número de disco (por ejemplo, `/dev/md1p1`).
- Para un %%XFS|xfs%% cifrado, use `/dev/mapper/mdXp1`.
- Para unidades fuera del %%array|array%%: `xfs_repair -v /dev/sdX1`
- Asegúrese de estar utilizando el identificador de dispositivo correcto.

:::warning
Running this command on an %%array|array%% disk outside of **%%Maintenance Mode|maintenance-mode%%** will invalidate %%parity|parity%%.
:::
