La actualización de Unraid se realiza usando la nueva **herramienta Actualizar OS** con una interfaz fácil de usar:

1. En el %%WebGUI|web-gui%%, haga clic en el menú desplegable de arriba a la derecha y seleccione **Buscar actualizaciones** o navegue a ***Herramientas → Actualizar OS***.
2. Elija el canal **Estable** para versiones estables o el canal **Siguiente** para betas y RCs.
3. Haz clic en **Ver Registro de Cambios para Comenzar la Actualización**. El Registro de Cambios aparecerá para que lo revises antes de presionar **Continuar**.
4. Haga clic en **Confirmar y comenzar actualización** para aplicar la última versión estable.
5. Cuando se le pida, reinicie su servidor para completar la actualización.

:::note

Es posible que deba iniciar sesión en su cuenta Unraid para acceder a las actualizaciones, especialmente para lanzamientos de la rama "Siguiente".

:::
