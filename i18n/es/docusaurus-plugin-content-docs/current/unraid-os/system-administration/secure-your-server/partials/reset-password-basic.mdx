Este método borra **todas las contraseñas de los usuarios**, incluyendo\*%%root|root-user%% y usuarios compartidos.

Para la forma más sencilla de restablecer tu contraseña:

1. **Apaga** tu servidor Unraid.
2. **Conecta el dispositivo USB de arranque** a una computadora (Windows o Mac).
3. **Elimina estos archivos** de la unidad USB:
   - `/config/shadow`
   - `/config/smbpasswd`
4. **Desconecta el dispositivo de arranque USB** y reconéctalo a tu servidor Unraid, luego enciéndelo.
5. **Crea una nueva contraseña de root** cuando te lo pida al iniciar.
