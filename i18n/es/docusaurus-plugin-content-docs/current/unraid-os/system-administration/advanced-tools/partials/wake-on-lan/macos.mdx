<h4>Método en Terminal (recomendado)</h4>

1. Instala `wakeonlan` usando Homebrew:

   ```
   brew install wakeonlan
   ```

2. Envía el paquete mágico:

   ```
   wakeonlan MAC_ADDRESS
   ```

<h4>Alternativa con GUI: miniWOL</h4>

1. Descarga [miniWOL v3](https://apps.apple.com/app/miniwol/id6474282023) desde la App Store.
2. Añade tu servidor usando su dirección MAC.
3. Haz clic en el icono del sol para despertar tu servidor.
