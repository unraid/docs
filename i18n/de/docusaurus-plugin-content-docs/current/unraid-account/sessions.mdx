---
title: Sitzungen
description: Wie Sie Kontositzungen in der Unraid Account-App überprüfen und widerrufen
---

Die Registerkarte **Sitzungen** zeigt aktuelle Browsersitzungen für Ihr Unraid-Konto an. Verwenden Sie sie, um die Sitzungsaktivität zu überprüfen und Sitzungen zu widerrufen, die Sie nicht mehr aktiv halten möchten.

Die Account-App löscht Sitzungsdatensätze routinemäßig, sobald sie nicht mehr benötigt werden.

## Bevor Sie beginnen

- Gehen Sie zu [account.unraid.net](https://account.unraid.net) und melden Sie sich bei Ihrem Unraid-Konto an.

## Den Tab „Sessions“ öffnen

1. Öffnen Sie die Unraid-Account-App.
2. Öffnen Sie **Kontoeinstellungen** und wählen Sie dann den Tab **Sessions** aus.

## Ihre Sessions überprüfen

Der Tab **Sessions** listet Ihre Kontositzungen in einer Tabelle auf.

Verwenden Sie die Tabelle, um Folgendes zu prüfen:

- **IP-Adresse**: die für die Session erfasste Netzwerkadresse.
- **User-Agent**: die für die Session erfassten Browser- oder Client-Informationen.
- **Aktivität**: wann die Session zuletzt aktiv war und wann sie erstellt wurde.
- **Läuft ab**: wann die Session abläuft, wenn sie nicht vorher widerrufen wird.
- **Widerrufen**: wann die Account-App die Session widerrufen hat, falls sie bereits widerrufen wurde.
- **Status**: ob die Session **Aktiv**, **Abgelaufen** oder **Widerrufen** ist.

Das Badge **Aktuell** markiert die Browser-Session, die Sie gerade verwenden.

Eine Schaltfläche **Widerrufen** wird nur bei Zeilen mit dem Status **Aktiv** angezeigt. Abgelaufene und widerrufene Sessions haben keine Aktionsschaltfläche.

## Eine andere aktive Session widerrufen

1. Suchen Sie die aktive Session, die Sie beenden möchten.
2. Klicken Sie in der Zeile dieser Session auf **Widerrufen**.
3. Warten Sie, bis die Account-App die Tabelle aktualisiert.

Wenn Sie eine andere aktive Session widerrufen, werden Sie nicht von der Browser-Session abgemeldet, die Sie gerade verwenden.

## Die aktuelle Session widerrufen

1. Suchen Sie die Session mit dem Badge **Aktuell**.
2. Klicken Sie auf **Widerrufen**.
3. Klicken Sie im Bestätigungsdialog auf **Session widerrufen**.

Die Account-App meldet Sie sofort ab, nachdem sie die aktuelle Session widerrufen hat.

## Alle Sessions widerrufen

1. Klicken Sie auf **Alle Sessions widerrufen**.
2. Klicken Sie im Bestätigungsdialog auf **Session widerrufen**.

Die Account-App widerruft alle aktiven Kontosessions, einschließlich der Browser-Session, die Sie gerade verwenden, und meldet Sie sofort ab.

## Fehlerbehebung

- Wenn die Tabelle nicht geladen wird, klicken Sie auf die Schaltfläche zum Aktualisieren und versuchen Sie es erneut.
- Wenn bei einer Session **Abgelaufen** oder **Widerrufen** angezeigt wird, ist sie nicht mehr aktiv und kann nicht erneut widerrufen werden.
