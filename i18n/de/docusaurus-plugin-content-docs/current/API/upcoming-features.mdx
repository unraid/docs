## Versionsunterstützung

| Unraid-Version      | API-Version       | Unterstützungsstatus |
| ------------------- | ----------------- | -------------------- |
| Unraid v7.2-beta.1+ | Aktuell           | ✅ Aktiv              |
| 7.0 - 7.1.x         | v4.x über Plugin  | ⚠️ Eingeschränkt     |
| 6.12.x              | v4.x über Plugin  | ⚠️ Eingeschränkt     |
| \< 6.12             | Nicht unterstützt | ❌ EOL                |

:::warning[Legacy Unterstützung]
Versionen vor Unraid 7.2 erfordern die Installation der API über das Unraid Connect-Plugin. Einige Funktionen sind in älteren Versionen möglicherweise nicht verfügbar.
:::

:::tip[Pre-release Versionen]
Sie können das Unraid Connect-Plugin immer installieren, um Vorabversionen der API zu erhalten und neuen Funktionen frühzeitig Zugang zu verschaffen, bevor sie in den Unraid OS-Veröffentlichungen enthalten sind.
:::
