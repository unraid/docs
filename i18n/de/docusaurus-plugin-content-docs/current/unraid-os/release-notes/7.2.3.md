## Änderungen vs. [7.2.2](7.2.2.md)

### Speicher

- Behebung: Samba wurde aktualisiert, um Probleme mit Time Machine zu lösen.
- Behebung: Precleared-Disksignatur wird erst nach einem Neustart erkannt.

### webGUI

- Behebung: Regression bei der Anwendung von Verläufen über Hintergrundbilder im Header.
- Behebung: Benachrichtigungstext wird manchmal in falschen Farben angezeigt, was das Lesen erschwert.
- Fix: Mögliche XSS-Probleme beim Testen der SMTP-Konfiguration unter _**Einstellungen → Benachrichtigungseinstellungen → SMTP-Einstellungen**_

### Verschiedenes

- Fix: DNS-Einstellungen könnten durch andere Geräte im Netzwerk überschrieben werden (avahidnsconfd deaktivieren)
- Fix: Readme wird in Docker-XML-Vorlagen nicht ausgefüllt
- Fix: Fehlende novnc-Dateien verursachen Warnungen im Syslog

### Unraid API

- dynamix.unraid.net 4.28.2 – [Änderungen ansehen](https://github.com/unraid/api/releases).

### Linux-Kernel

- Version 6.12.54-Unraid (keine Änderungen)

### Basis-Distro-Updates

- samba: Version 4.23.4
