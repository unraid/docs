# Version 7.2.2 2025-11-21

Wenn Sie bereits auf Version 7.2.1 sind, ist dies ein **optionales** Update. Wir haben es in erster Linie für Testnutzer veröffentlicht, um einen Fehler zu beheben, der sie daran hinderte, das Ablaufdatum ihres Testzeitraums zu sehen. Wir haben auch ein Problem im Zusammenhang mit benutzerdefinierten Hintergrundheaderfarben behoben; wir empfehlen ein Update, wenn Sie diese Einstellung verwenden.

Wenn Sie Version 7.2.0 oder früher verwenden, können Sie direkt auf 7.2.2 aktualisieren.

## Upgrade

Für Schritt-für-Schritt-Anleitungen siehe [Unraid aktualisieren](../../unraid-os/system-administration/maintain-and-update/upgrading-unraid.mdx).

### Bekannte Probleme

Für weitere bekannte Probleme siehe die [7.2.1 Release Notes](7.2.1.md#known-issues).

### Rückschritte

Wenn Sie auf eine Version vor 7.2.1 zurückgehen, schauen Sie sich auch die [7.2.1 Release Notes](7.2.1.md#rolling-back) an.
