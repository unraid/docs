1. In den Freigabeeinstellungen **Primärspeicher** auf *Cache* und **Sekundärspeicher** auf *Array* setzen.
2. Die **Mover-Aktion** auf *Cache -> Array* setzen.
3. Der **%%Mover|mover%%** läuft automatisch gemäß Ihrem Zeitplan, um Dateien zu übertragen. Manuelle %%Mover|mover%% Ausführungen sind möglich, aber nicht erforderlich.
