Wenn Sie einen Docker-Container oder ein Plugin neu installieren müssen (zum Beispiel, wenn Ihre `docker.img`-Datei beschädigt ist), vereinfacht "Community Applications" dies für Sie. Alle Anwendungseinstellungen werden als Docker-Template-XML-Dateien in `/boot/config/plugins/dockerMan/templates-user` gespeichert. Weitere Informationen zum Docker Template XML-Schema und benutzerdefinierten Tags finden Sie in der [Docker Template XML Schema-Dokumentation](https://forums.unraid.net/topic/38619-docker-template-xml-schema/) in den Unraid-Foren.

Um eine Anwendung erneut zu installieren:

1. Gehen Sie zum **Apps**-Tab in der %%WebGUI|web-gui%%.
2. Wählen Sie den **Vorherige Apps**-Filter.
3. Finden Sie die Anwendung, die Sie erneut installieren möchten, und folgen Sie den Anweisungen.

Ihre gespeicherten Einstellungen werden automatisch wiederhergestellt, was eine schnelle und einfache Wiederherstellung ermöglicht.
