1. In ***Einstellungen → VM-Manager***, einen Standard-%%VirtIO|virtio%%-ISO-Pfad festlegen (z.B. `/mnt/user/isos/virtio-win.iso`).
2. Beim Erstellen einer Windows-%%VM|vm%% wird das ISO automatisch als virtuelles CD-ROM angehangen.
3. Während der Windows-Installation Treiber von der %%VirtIO|virtio%% ISO laden, wenn nach Speichercontrollern gefragt wird.
