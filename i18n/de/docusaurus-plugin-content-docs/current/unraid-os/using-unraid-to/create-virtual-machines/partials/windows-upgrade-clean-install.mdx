1. Laden Sie das [Windows 11 ISO](https://www.microsoft.com/en-us/software-download/windows11) herunter.
2. Laden Sie das ISO in Ihre `isos`-Freigabe auf Ihrem Unraid-Server hoch.
3. Ändern Sie Ihre %%VM|vm%%-Vorlage, um von dem ISO in der `isos`-Freigabe zu starten.
4. Starten Sie die %%VM|vm%% vom ISO und führen Sie die Neuinstallation durch.
5. Installieren Sie Ihre Anwendungen neu und stellen Sie Ihre Daten aus dem Backup wieder her.
