%%IOMMU|iommu%% (Ein-/Ausgabe-Speicherverwaltungseinheit, auch bekannt als Intel VT-d oder AMD-Vi) ermöglicht sicheres und effizientes Durchschleifen von Geräten zu %%VMs|vm%%.

- Erforderlich, um PCIe-Geräte (GPUs, USB-Controller, NVMe-Laufwerke) direkt einem %%VM|vm%% zuzuweisen.
- Bietet Speicherisolierung und Schutz, um zu verhindern, dass Geräte auf nicht autorisierte Speicherbereiche zugreifen.
- Wesentlich für %%GPU-Durchleitung|gpu-passthrough%%, erweiterte Netzwerke und leistungsstarke Arbeitslasten.
