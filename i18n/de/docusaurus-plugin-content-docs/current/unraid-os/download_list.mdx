# Versionsarchiv

Verwenden Sie dieses Archiv, um Versionshinweise und Download-Links für Unraid OS-Versionen zu finden. Um einen vorhandenen Server zu aktualisieren, verwenden Sie den integrierten Updater; siehe den [Standard-Aktualisierungsprozess](/unraid-os/system-administration/maintain-and-update/upgrading-unraid#standard-update-process) für die einzelnen Schritte. Um ein neues Boot-Gerät zu erstellen, verwenden Sie den [Unraid USB Creator](https://unraid.net/download).

Das Archiv gruppiert Releases nach Supportstatus. Nur der neueste Patch oder Pre-Release-Build in einer unterstützten Minor-Serie erhält ggf. Updates; ältere Builds in derselben Serie werden ersetzt. Als **EOL** gekennzeichnete Releases werden nicht unterstützt und erhalten keine Updates mehr.

Wenn Sie sich entscheiden, welche Release Sie installieren möchten, beginnen Sie mit [Unraid OS release types](./updating-unraid/release-types.mdx). Zu den Details zur Lizenzberechtigung siehe die [Licensing FAQ](./troubleshooting/licensing-faq.mdx#no-extension).
