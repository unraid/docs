Stromausfälle sind einer der Hauptgründe für unsaubere Abschaltungen. Schützen Sie Ihr System mit einer richtig konfigurierten USV, die Unraid automatisch herunterfahren kann, bevor der Akku leer ist.

:::note

Unraid unterstützt die meisten USV-Einheiten über das %%apcupsd-Protokoll|apcupsd%% (APC und CyberPower sind normalerweise kompatibel). Wenn Ihre USV nicht unterstützt wird, ziehen Sie in Betracht, das Network UPS Tools (NUT) Plugin von Community Applications zu verwenden.

:::
