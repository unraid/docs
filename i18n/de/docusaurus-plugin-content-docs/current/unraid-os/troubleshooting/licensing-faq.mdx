## Einkauf

### Besitze ich meine Softwarelizenz? {/* #own-license */}

Wenn Sie [eine Unraid OS-Lizenz erwerben](https://unraid.net/pricing), besitzen Sie eine zeitlich unbegrenzte Kopie der Software. Ihre Lizenz ist unbegrenzt gültig und läuft nicht ab, selbst wenn Sie nicht für zukünftige Updates zahlen.

### Lizenzverwaltung

#### Installation per WebGUI-Download

1. Navigieren Sie zu [account.unraid.net/keys](https://account.unraid.net/keys), wählen Sie neben Ihrem Schlüssel **View options** aus und klicken Sie dann auf **Copy Key URL**.
2. Ziehen Sie die Registrierungsschlüsseldatei in das `config`-Verzeichnis.
3. In der %%WebGUI|web-gui%% **Stoppen** Sie das %%array|array%%, und **Starten** Sie das %%array|array%% erneut, um den neuen Schlüssel anzuwenden.
4. Fügen Sie die URL in das Feld **Key file URL** ein und klicken Sie auf **Install**.

#### Wie kann ich meinen Registrierungstyp bestimmen?

1. Wenn Ihr Server läuft und das Startvolume in Ihrem Netzwerk freigegeben ist, öffnen Sie unter **Network** die Freigabe `flash` (Boot-Gerät; bei den meisten Installationen derselbe Freigabename).
2. Ziehen Sie die Registrierungsschlüsseldatei in das `config`-Verzeichnis.
3. In der %%WebGUI|web-gui%% **Stoppen** Sie das %%array|array%%, und **Starten** Sie das %%array|array%% erneut, um den neuen Schlüssel anzuwenden.

#### Installation per USB-Bootmedium

Dies gilt nicht, wenn Sie internen Boot konfiguriert haben.

1. Stellen Sie sicher, dass Sie ein aktuelles Backup Ihres USB-Laufwerks haben. Verwenden Sie [Unraid Connect](../../unraid-connect/overview-and-setup.mdx) (empfohlen) oder die lokale Backup-Option unter ***Main → Flash → Flash Backup***.
2. Fahren Sie Ihren Unraid-Server herunter und entfernen Sie das USB-Startgerät.
3. Schließen Sie das USB-Laufwerk an einen anderen Computer an.
4. Öffnen Sie das USB-Laufwerk und kopieren Sie Ihre `.key` Datei in den `/config` Ordner.
   *Stellen Sie sicher, dass dies die einzige `.key` Datei ist - löschen Sie alle anderen.*
5. Werfen Sie das USB-Laufwerk sicher aus, installieren Sie es wieder in Ihrem Server und starten Sie ihn neu.

### Wie kann ich meinen Registrierungstyp bestimmen? {/* #registration-type */}

Navigieren Sie zu ***Werkzeuge → Registrierung*** im %%WebGUI|web-gui%%. Dort finden Sie Ihren aktuellen Lizenztyp und Registrierungsdetails.

Bei Fragen zum internen Boot siehe die [FAQ zu internem Boot (7.3+)](../getting-started/set-up-unraid/internal-boot-faq.mdx).
Bei Fragen zur TPM-basierten Lizenzierung siehe die [FAQ zur TPM-Lizenzierung (7.3+)](tpm-licensing-faq.mdx).

## Lizenztypen & Funktionen

### Ist Unraid OS ein Abonnement? {/* #subscription */}

Nein. Unraid OS ist eine **zeitlich unbegrenzte Lizenz**:

- **Starter** und **Unleashed** beinhalten ein Jahr lang Updates, danach können Sie eine jährliche Verlängerungsgebühr von 36 $ USD zahlen (optional).
- **Lifetime** beinhaltet Updates für die gesamte Produktlebensdauer.
- Wenn Sie sich entscheiden, die Verlängerungsgebühr nicht zu bezahlen, behalten Sie Ihre bestehende Version auf unbestimmte Zeit; Sie erhalten einfach keine neuen Hauptupdates.

### Was passiert, wenn ich meine Starter- oder Unleashed-Lizenz nicht verlängere? {/* #no-extension */}

- Sie behalten Ihre Lizenz und können Ihre aktuelle Version des Unraid OS auf unbestimmte Zeit verwenden.
- Sie erhalten keine neuen Funktionsupdates oder großen Versionsupgrades.
- Sie bleiben berechtigt, Patch-Releases und Sicherheitsupdates innerhalb derselben Minor-Version zu erhalten (z.B. 7.1.x, wenn Ihre Lizenz bei 7.1.0 abgelaufen ist).
- Sobald eine neue Minor-Version veröffentlicht wird (z.B. 7.2.0), werden nur Sicherheitsupdates für die vorherige Minor-Version bereitgestellt.
- Wenn eine Version das Ende ihres Lebenszyklus (EOL) erreicht, werden keine weiteren Updates bereitgestellt.
- Sie können die Verlängerungsgebühr jederzeit zahlen, um wieder Zugriff auf die neuesten Updates zu erhalten.

## Fehlerbehebung

### Was passiert, wenn mein Startgerät ausfällt? Muss ich eine Lizenz erneut kaufen? {/* #usb-failure-license */}

Registrierungsschlüssel für Unraid-Testversionen sind 30 Tage gültig und können zweimal um jeweils 14 zusätzliche Tage verlängert werden. Nach zwei Testphasenverlängerungen können keine weiteren Verlängerungen gewährt werden. Um eine Verlängerung zu beantragen, müssen Sie das Array stoppen, zur **Werkzeuge > Registrierung** Seite navigieren und auf "Verlängerung anfordern" klicken.

Nein, Sie müssen Ihre Lizenz nicht neu kaufen, wenn Ihr USB-Flash-Gerät ausfällt.

1. Bereiten Sie neue, hochwertige [Bootmedien](../getting-started/set-up-unraid/create-your-bootable-media.mdx) vor (zum Beispiel einen USB-Flash-Laufwerk).
2. Installieren Sie das Unraid-Betriebssystem auf dem neuen Gerät mittels USB-Flash-Creator oder einer manuellen Methode.
3. Starten Sie Ihren Server mit dem neuen Boot-Gerät.
4. Gehen Sie zu ***Werkzeuge → Registrierung*** im %%WebGUI|web-gui%%.
5. Klicken Sie auf **Schlüssel Ersetzen** und folgen Sie den Anweisungen, um Ihre Lizenz auf das neue Gerät zu übertragen.

Die erste Übertragung kann jederzeit erfolgen, während nachfolgende Übertragungen einmal alle 12 Monate über das automatische System erlaubt sind. Wenn Sie Ihre Lizenz vor Ablauf der 12 Monate erneut übertragen müssen, wenden Sie sich an den Unraid-Support mit den alten und neuen USB-%%GUID|guid%%s für manuelle Unterstützung.

### Was soll ich tun, wenn ich beim Registrieren meines Startgeräts einen Fehler erhalte: '####-####-####-#############' ist bereits einem anderen Benutzer registriert? {/* #guid-error */}

Dieser Fehler zeigt an, dass Ihr Boot-Gerät keine eindeutige Hardware-ID (%%GUID|guid%%) hat, wodurch es nicht mit Unraid OS registriert werden kann. Verwenden Sie einen anderen USB-Flash-Laufwerk. Hinweise zur Auswahl eines Laufwerks (einschließlich Fälschungen und Markenempfehlungen) finden Sie unter [Create your bootable media](../getting-started/set-up-unraid/create-your-bootable-media.mdx) und [Selecting a replacement device](../system-administration/maintain-and-update/changing-the-flash-device.mdx#selecting-a-replacement-device).

## Testlizenzen

### Wie funktionieren Unraid-Testversionen? {/* #trial-license */}

- Testlizenzen dauern 30 Tage und bieten die volle Unraid-Funktionalität ohne Speichergerätelimit.
- Sie benötigen qualitativ hochwertige USB-Bootmedien (typischerweise einen USB-Flash-Laufwerk) und das Unraid USB Flash Creator-Tool.
- Konsultieren Sie den [Leitfaden „Erste Schritte“](../getting-started/set-up-unraid/create-your-bootable-media.mdx) für Servereinstellungen.
- Testlizenzen erfordern beim Start eine Internetverbindung zur anfänglichen Validierung.
- Wenn Ihre Testversion abläuft, wird das Array angehalten. Sie können dann einen Lizenzschlüssel kaufen oder eine Verlängerung der Testversion beantragen.
- Sie können Ihren Test um 15 Tage verlängern, wenn Sie mehr Zeit für die Bewertung von Unraid benötigen. Sie können dies zweimal tun, insgesamt 60 Tage (Details unten).

### Wie funktionieren Unraid-Testversionen? {/* #trial-license */}

Registrierungsschlüssel für Unraid-Testversionen sind 30 Tage gültig und können zweimal um jeweils 14 zusätzliche Tage verlängert werden. Nach zwei Testphasenverlängerungen können keine weiteren Verlängerungen gewährt werden. Um eine Verlängerung zu beantragen, müssen Sie das Array stoppen, zur **Werkzeuge > Registrierung** Seite navigieren und auf „Verlängerung anfordern“ klicken.

Wenn die beiden zusätzlichen Verlängerungen nicht verwendet wurden und diese Option bei Ihnen nicht angezeigt wird oder aus irgendeinem Grund nicht funktioniert, senden Sie uns bitte die GUID des Boot-Geräts aus **Tools > Registration** an [contact@unraid.net](mailto:contact@unraid.net) oder über unser Support-Portal unter [support.unraid.net](https://support.unraid.net/support/home), und wir helfen Ihnen gerne weiter.
