1. Stellen Sie sicher, dass Sie ein aktuelles Backup Ihres USB-Laufwerks haben. Verwenden Sie [Unraid Connect](../../../../unraid-connect/overview-and-setup.mdx) (empfohlen) oder die lokale Sicherungsoption unter ***Main → Boot Device → Boot Device Backup*** (bei Versionen vor 7.3.0 finden Sie sie unter **Flash** / **Flash device**).
2. Fahren Sie Ihren Unraid-Server herunter und entfernen Sie das USB-Startgerät.
3. Schließen Sie das USB-Laufwerk an einen anderen Computer an.
4. Öffnen Sie das USB-Laufwerk und kopieren Sie Ihre `.key` Datei in den `/config` Ordner.
   *Stellen Sie sicher, dass dies die einzige `.key` Datei ist - löschen Sie alle anderen.*
5. Werfen Sie das USB-Laufwerk sicher aus, installieren Sie es wieder in Ihrem Server und starten Sie ihn neu.
