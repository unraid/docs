*Verwenden Sie diese Methode, wenn das USB-Flash-Erstellungstool auf Ihrem System nicht funktioniert.*

:::important[Before Sie beginnen]

Verwenden Sie die manuelle Methode nur, wenn Sie mit grundlegender Dateiverwaltung vertraut sind und Ihr System den USB Flash Creator nicht unterstützt.

:::

1. Bereiten Sie neue Boot-Medien nach der [manuellen Installationsmethode](../../../getting-started/set-up-unraid/create-your-bootable-media.mdx#manual-install-method) vor.
2. Kopieren Sie Ihren gesicherten `config`-Ordner auf das neue Boot-Gerät und überschreiben Sie dabei vorhandene Dateien.
3. Fahren Sie den Server herunter, tauschen Sie die Boot-Geräte aus und schalten Sie ihn wieder ein.
4. Wenn Sie mit `Ungültiger, fehlender oder abgelaufener Registrierungsschlüssel` aufgefordert werden, wählen Sie **Registrierungsschlüssel**.

<div
style={{
margin: "auto",
maxWidth: "600px",
display: "flex",
flexDirection: "column",
alignItems: "center",
}}
>
![Ungültiger Schlüssel](/img/Invalidkey.png)
</div>

5. Wählen Sie in ***Werkzeuge → Registrierung*** **Schlüssel ersetzen** und geben Sie Ihre E-Mail-Adresse ein.

<div
style={{
margin: "auto",
maxWidth: "600px",
display: "flex",
flexDirection: "column",
alignItems: "center",
}}
>
![Ungültiger Schlüssel](/img/Replacekey.png)
</div>

6. Folgen Sie den Anweisungen in der E-Mail, um Ihren neuen Schlüssel zu installieren.
7. Wählen Sie **Fertig**.
