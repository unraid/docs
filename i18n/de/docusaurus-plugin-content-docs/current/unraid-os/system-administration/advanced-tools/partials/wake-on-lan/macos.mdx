<h4>Terminal-Methode (empfohlen)</h4>

1. Installieren Sie `wakeonlan` mit Homebrew:

   ```
   brew install wakeonlan
   ```

2. Senden Sie das magische Paket:

   ```
   wakeonlan MAC_ADDRESS
   ```
