1. Wählen Sie das Auge-Symbol ![Peer-Konfiguration](/img/eye-icon.png), um die Peer-Konfiguration anzuzeigen.

<div style={{ margin: 'auto', maxWidth: '600px', display: 'flex', flexDirection: 'column', alignItems: 'center' }}>

2. In der %%WireGuard|wireguard%%-Mobil-App wählen Sie **Aus QR-Code erstellen** und scannen Sie den QR-Code.
3. Benennen Sie die Verbindung und verbinden Sie sich. Der %%VPN-Tunnel|vpn-tunnel%% sollte schnell starten.
4. Halten Sie den QR-Code privat - jeder, der ihn hat, kann auf Ihr VPN zugreifen.
