# バージョンアーカイブ

このアーカイブを使用して、Unraid OS の各バージョンのリリースノートとダウンロードリンクを確認できます。既存のサーバーを更新するには、内蔵のアップデーターを使用してください。手順については、[標準の更新プロセス](./updating-unraid/index.mdx#standard-update-process) を参照してください。新しい起動デバイスを作成するには、[Unraid USB Creator](https://unraid.net/download) を使用してください。

アーカイブでは、リリースはサポート状況ごとにグループ化されています。サポート対象のマイナー系列では、最新のパッチまたはプレリリースビルドのみが、適切な更新の対象となります。同じ系列の古いビルドは、それに置き換えられます。**EOL** と表示されたリリースはサポート対象外であり、更新は提供されません。

どのリリースをインストールするかを決める場合は、まず [Unraid OS release types](./updating-unraid/release-types.mdx) をご覧ください。ライセンスの対象条件については、[Licensing FAQ](./troubleshooting/licensing-faq.mdx#no-extension) を参照してください。
