# リリースノート · Unraid OS 7.2.7

## バージョン 7.2.7 2026-05-21

このリリースでは、セキュリティ更新、Docker の更新、WebGUI の修正、およびコアシステムコンポーネントのパッケージ更新に重点を置いています。

セキュリティ上重要な環境では、アップグレードを推奨します。

## アップグレード

手順については、[Unraid の更新](/unraid-os/updating-unraid/) をご覧ください。[ライセンス](/unraid-os/troubleshooting/licensing-faq/) に関するご質問がありますか?

## [7.2.6](/unraid-os/release-notes/7.2.6/) との差分

### セキュリティ

- docker、nginx、glibc、jq、avahi、および glibc サポートライブラリのセキュリティ更新。

### コンテナ / Docker

- Docker 29.5.1
- Docker VLAN コンテナは、更新後も設定されたゲートウェイを保持します。
- Docker コンテナのメニューは、多数のコンテナがあるページでも引き続きアクセス可能です。

### WebGUI / システム

- サーバープラグインで Connect のバージョンチェックが復元されました。

### Linux カーネル

- カーネルのアップグレード: 6.12.90-Unraid (CVE-2026-43490、CVE-2026-46333)
- Docker 有効システムおよび一部の USB 3.1 デバイスに影響するパフォーマンス低下の修正。
