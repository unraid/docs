# TPMライセンスFAQ（7.3+）

このページには、Unraid OS における TPM ベースのライセンスについてのよくある質問が掲載されています。一般的なライセンスに関する質問は、[ライセンス FAQ](/unraid-os/troubleshooting/licensing-faq/) を参照してください。内部ブートに関する質問は、[内部ブート FAQ（7.3+）](/unraid-os/getting-started/set-up-unraid/internal-boot-faq/) を参照してください。

---

### フラッシュベースのライセンスから TPM ベースのライセンスへ移行するにはどうすればよいですか？ {/* #tpm */}

:::note[Unraid 7.3以降]

TPM ベースのライセンスは、Unraid 7.3 以降で利用できます。

:::

新規キーおよび交換キーは、可能な場合はすべてTPMベースのライセンスを使用します。これはフラッシュベースのライセンスよりも信頼性が高いためです。

TPM ベースのライセンスへ事前に切り替えるには、***Tools → Registration*** に移動し、**Move License to TPM** をクリックします。Unraid Account のウェブサイトに移動したら、置き換え要求を承認できます。

**Move License to TPM** をクリックすると、Unraid Account のフローで **Replace License Key** 画面が表示されます:

![Move License to TPM をクリックした後に表示される Replace License Key 画面](/img/move-license-to-tpm-replace-key.png)

**Move License to TPM** ボタンが利用できない場合、Unraid は TPM 2.0 デバイスを検出できませんでした。BIOS で無効になっている可能性があります。**Trusted Computing** に関連する設定を確認し、**Secure Boot** は無効のままにしてください。

### TPM ベースのライセンスへ移行する利点は何ですか？ {/* #tpm-benefits */}

従来の USB ブートデバイスは、一部のユーザーにとって信頼性が低くなりつつあります。TPM ベースのライセンスなら、起動ハードウェアを変更してもライセンスは有効なままです。

### 新しいマザーボードに交換した場合はどうなりますか？ {/* #tpm-new-motherboard */}

新しいマザーボードでは、TPM ベースのライセンスを交換できます。これは、新しい USB デバイスでフラッシュベースのライセンスを交換する場合と同様です。この場合、ブラックリストに登録されるのは元のライセンスのみで、マザーボード全体ではありません。

### BIOS を更新したらどうなりますか？新しいライセンスは必要ですか？ {/* #tpm-bios */}

TPM は BIOS の更新の影響を受けるべきではありません。もし何らかの理由で影響を受けた場合は、[新しいマザーボード](#tpm-new-motherboard) に交換した場合と同様にライセンスを交換できます。

### マザーボードに TPM デバイスがない場合はどうなりますか？ {/* #tpm-missing */}

TPM ベースのライセンスへの移行は必須ではありませんが、古いマザーボードの多くには TPM アドオンデバイスを取り付けられるヘッダーがあります。これらのデバイスは安価で、広く入手可能です。

### フラッシュから起動している間に TPM ベースのライセンスへ切り替えられますか？ {/* #tpm-flash-boot */}

はい、ライセンス方式は起動方式に依存しません。

内部ブートに固有の質問については、[内部ブート FAQ（7.3+）](/unraid-os/getting-started/set-up-unraid/internal-boot-faq/) を参照してください。

### Unraid VM 内から TPM を使用できますか？ {/* #tpm-vm */}

いいえ、これはサポートされていません。

### ハードウェア障害からの復旧が必要な場合、ライセンスを TPM と USB の間で移行できますか？ {/* #tpm-usb-replacement */}

はい.

TPM ベースのライセンスへ移行すると、元の USB デバイス上のライセンスはブラックリストに登録されます。そこからは:

- ライセンス置換フローを使用して USB デバイスに戻すことができます。
- TPM 搭載マザーボードから別の TPM 搭載マザーボードへは、TPM 置換フローを使用して移行できます。

通常の置換フローを妨げるハードウェア障害に遭遇した場合は、[Unraid サポート](https://support.unraid.net/support/home) に連絡して支援を受けてください。

### TPM のリセットは Unraid ライセンスに影響しますか？ {/* #tpm-reset */}

いいえ。Unraid の TPM ライセンスは TPM 識別子を使用するため、TPM のリセットは Unraid ライセンスに影響しません。
