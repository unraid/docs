*Use this method if the USB Flash Creator tool does not work for your system.*

:::important[Before 始める]

Use the manual method only if you are comfortable with basic file management and your system does not support the USB Flash Creator.

:::

1. Prepare new boot media following the [Manual install method](../../../getting-started/set-up-unraid/create-your-bootable-media.mdx#manual-install-method).
2. Copy your backed-up `config` folder to the new boot device, overwriting existing files.
3. Shut down the server, swap the boot devices, and power on.
4. `Invalid, missing or expired registration key` と表示されたら、**Registration key** を選択します。

<div style={{ margin: 'auto', maxWidth: '600px', display: 'flex', flexDirection: 'column', alignItems: 'center' }}>
  ![Invalid Key](/img/Invalidkey.png)
</div>

5. ***Tools → Registration*** で、**Replace key** を選択し、メールアドレスを入力します。

<div style={{ margin: 'auto', maxWidth: '600px', display: 'flex', flexDirection: 'column', alignItems: 'center' }}>
  ![Invalid Key](/img/Replacekey.png)
</div>

6. メールに記載された手順に従って、新しい key をインストールします。
7. **Done** を選択します。
