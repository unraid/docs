1. Installez le plugin [Unraid Connect](../../unraid-connect/overview-and-setup.mdx) à partir de [Community Applications](../../unraid-os/using-unraid-to/run-docker-containers/community-applications.mdx).
2. Pas de connexion à Unraid Connect requise pour l'accès local à l'API.
3. Configurer les paramètres du plugin.
4. Activez le bac à sable %%GraphQL|graphql%%.
5. Commencez à explorer l'API !
