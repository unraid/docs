---
title: Sessions
description: Comment consulter et révoquer les sessions du compte dans l’application Unraid Account
---

L’onglet **Sessions** affiche les sessions de navigateur récentes pour votre compte Unraid. Utilisez-le pour examiner l’activité des sessions et révoquer celles que vous ne souhaitez plus garder actives.

L’application Account efface régulièrement les enregistrements de session lorsqu’ils ne sont plus nécessaires.

## Avant de commencer

- Rendez-vous sur [account.unraid.net](https://account.unraid.net) et connectez-vous à votre compte Unraid.

## Ouvrez l’onglet Sessions.

1. Ouvrez l’application Unraid Account.
2. Ouvrez **Paramètres du compte**, puis sélectionnez l’onglet **Sessions**.

## Consultez vos sessions.

L’onglet **Sessions** répertorie vos sessions de compte dans un tableau.

Utilisez le tableau pour vérifier :

- **Adresse IP** : l’adresse réseau enregistrée pour la session.
- **Agent utilisateur** : les informations du navigateur ou du client enregistrées pour la session.
- **Activité** : la dernière fois où la session a été détectée et sa date de création.
- **Expiration** : quand la session expire si elle n’est pas révoquée avant.
- **Révoquée** : quand l’application Account a révoqué la session, si elle a déjà été révoquée.
- **Statut** : si la session est **Active**, **Expirée** ou **Révoquée**.

Le badge **Actuelle** indique la session de navigateur que vous utilisez maintenant.

Un bouton **Révoquer** apparaît uniquement sur les lignes dont le statut est **Active**. Les sessions expirées et révoquées n’ont pas de bouton d’action.

## Révoquer une autre session active

1. Trouvez la session active que vous souhaitez terminer.
2. Cliquez sur **Révoquer** dans la ligne de cette session.
3. Attendez que l’application Account actualise le tableau.

La révocation d’une autre session active ne vous déconnecte pas de la session de navigateur que vous utilisez maintenant.

## Révoquer la session actuelle

1. Trouvez la session portant le badge **Actuelle**.
2. Cliquez sur **Révoquer**.
3. Dans la boîte de dialogue de confirmation, cliquez sur **Révoquer la session**.

L’application Account vous déconnecte immédiatement après avoir révoqué la session actuelle.

## Révoquer toutes les sessions

1. Cliquez sur **Révoquer toutes les sessions**.
2. Dans la boîte de dialogue de confirmation, cliquez sur **Révoquer la session**.

L’application Account révoque chaque session active du compte, y compris la session de navigateur que vous utilisez maintenant, et vous déconnecte immédiatement.

## Dépannage

- Si le tableau ne se charge pas, cliquez sur le bouton d’actualisation et réessayez.
- Si une session affiche **Expirée** ou **Révoquée**, elle n’est plus active et ne peut pas être révoquée à nouveau.
