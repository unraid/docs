---
title: Gestion du serveur d'application de compte
description: Comment effectuer des actions sur les clés de licence et des mises à jour du système d'exploitation pour votre(vos) serveur(s)
---

:::info\[Prerequisite]

Un minimum de Unraid 6.12.8 ou la dernière version du plugin Unraid Connect.

:::

## Comment gérer mon serveur Unraid dans l'application de compte ?

Pour gérer votre serveur Unraid :

1. Allez à votre serveur Unraid
2. Ouvrez le menu déroulant en haut à droite et cliquez sur "Gérer le compte Unraid.net" ou sur tout bouton d'action lié à la clé de licence visible.

## Que faire si j'ai plusieurs serveurs ?

- Un seul serveur peut être géré dans cette application à la fois.
- Suivez les mêmes étapes ci-dessus pour tout serveur afin de remplacer le serveur montré dans cette application de compte.

## Comment retirer mon serveur de cette application ?

Il existe trois options :

1. Fermez votre navigateur Web. Cela réinitialise votre session.
2. Déconnectez-vous de votre compte.
3. Cliquez sur "Retirer le serveur" sur la carte de gestion du serveur.
