1. Dans ***Paramètres → Gestionnaire de VM***, définissez un chemin ISO %%VirtIO|virtio%% par défaut (ex. : `/mnt/user/isos/virtio-win.iso`).
2. Lors de la création d'une %%VM|vm%% Windows, l'ISO sera automatiquement attaché en tant que CD-ROM virtuel.
3. Lors de l'installation de Windows, chargez les pilotes depuis l'ISO %%VirtIO|virtio%% quand il vous sera demandé pour les contrôleurs de stockage.
