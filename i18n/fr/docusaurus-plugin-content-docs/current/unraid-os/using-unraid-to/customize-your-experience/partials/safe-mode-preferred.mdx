Préféré, car il ne nécessite même pas d'écran connecté à votre serveur.

1. Dans le **%%WebGUI|web-gui%%**, allez à ***Main → Array Operation***.
2. Cochez la case **Reboot en mode sans échec**.
3. Cliquez sur **Reboot** pour redémarrer votre serveur directement en mode sans échec - aucun clavier ou moniteur requis.
