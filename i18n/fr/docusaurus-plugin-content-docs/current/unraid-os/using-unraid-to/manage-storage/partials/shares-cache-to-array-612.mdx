1. Dans les paramètres du partage, définissez **Stockage Principal** sur *Cache* et **Stockage Secondaire** sur *Array*.
2. Définissez l'**action du Mover** sur *Cache -> Array*.
3. Le **%%Déplaceur|mover%%** fonctionne automatiquement selon votre calendrier pour transférer des fichiers. Les opérations manuelles du %%Déplaceur|mover%% sont possibles mais pas nécessaires.
