**Message :** `Cette version d'Unraid a été retirée.`

Si vous voyez ce message, cela signifie que vous utilisez une version beta ou candidate d'Unraid qui n'est pas activée pour une utilisation régulière.

Pour résoudre cela :

1. Ouvrez Unraid.
2. Allez à ***Tools → Mettre à jour le système***.
3. Installez la dernière version stable.

Une fois la mise à jour terminée, redémarrez votre serveur pour démarrer votre ensemble.
