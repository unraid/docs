:::info[Common cas d'utilisation]

Déplacer des fichiers de la %%cache pool|cache-pool%% vers l'%%array|array%% avant d'effectuer la maintenance ou des mises à niveau pour assurer la sécurité de vos données.

:::

Pour déplacer des fichiers de votre pool à l'%%array|array%% :

1. Allez dans **Paramètres** et désactivez Docker et %%VM|vm%% Manager. Cela empêche tout fichier d'être maintenu ouvert, permettant au %%Mover|mover%% de tout transférer sans encombre.
2. Dans l'onglet **Partages**, pour chaque partage que vous souhaitez déplacer (comme `appdata` ou `système`), définissez **Stockage principal** sur le pool de cache source %%cache pool|cache-pool%% et **Stockage secondaire** sur le %%array|array%%. Définissez **Action du Mover** sur **cache → array**.
3. Allez dans l'onglet **Principal** et cliquez sur **Déplacer maintenant** pour transférer les fichiers du %%cache pool|cache-pool%% vers le %%array|array%%.
4. Après que le %%Mover|mover%% ait terminé, vérifiez que les fichiers ont été déplacés en cliquant sur l'icône de dossier à côté de l'entrée de cache dans l'onglet **Principal**.
5. Une fois que tous les fichiers sont sur le %%array|array%%, vous pouvez rallumer en toute sécurité les services Docker et %%VM|vm%%.
