1. **Désactivez les services Docker et %%VM|vm%%** comme décrit ci-dessus.
2. Utilisez les commandes `rsync` ou `mv` pour transférer des fichiers entre les points de montage (par exemple, `/mnt/pool1/share` vers `/mnt/pool2/share`).
3. Vérifiez les fichiers avant de les supprimer du pool d'origine.
4. Réactivez les services Docker et %%VM|vm%%.

:::caution
Vérifiez toujours vos chemins de fichiers et utilisez les noms de partage corrects pour éviter toute perte de données.
:::
