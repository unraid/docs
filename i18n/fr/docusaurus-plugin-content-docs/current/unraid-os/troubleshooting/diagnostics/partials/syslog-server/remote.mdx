1. Réglez **Serveur syslog local** sur *Activé*.
2. Entrez l'adresse IP de votre serveur syslog sous **Serveur syslog distant**.
3. Cliquez sur **Appliquer**.
   - Les journaux seront diffusés vers l'appareil que vous avez spécifié.
