Les interruptions de courant sont l'une des principales raisons des arrêts non propres. Protégez votre système avec un onduleur configuré correctement qui peut arrêter automatiquement Unraid avant que la batterie ne se décharge.

:::note

Unraid prend en charge la plupart des onduleurs utilisant le protocole %%apcupsd Protocol|apcupsd%% (APC et CyberPower sont généralement compatibles). Si votre onduleur n'est pas pris en charge, envisagez d'utiliser le plugin Network UPS Tools (NUT) des Applications Communautaires.

:::
