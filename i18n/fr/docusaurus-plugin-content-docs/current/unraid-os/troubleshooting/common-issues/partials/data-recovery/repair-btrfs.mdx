- Utilisez `btrfs scrub` dans le %%WebGUI|web-gui%% pour rechercher et réparer certaines erreurs.
- Sur les disques uniques, le scrubbing peut détecter mais ne pas corriger certaines erreurs ; si c'est le cas, envisagez de copier les données et de reformater le disque ou le pool.

:::note
- Les outils de réparation actuels pour %%BTRFS|btrfs%% peuvent ne pas corriger toutes les corruptions ; consultez la documentation Unraid pour des outils mis à jour si nécessaire.
- Pour plus de détails sur le scrubbing et la récupération, reportez-vous à la documentation respective %%BTRFS|btrfs%% ou aux forums Unraid pour obtenir des conseils.
:::
