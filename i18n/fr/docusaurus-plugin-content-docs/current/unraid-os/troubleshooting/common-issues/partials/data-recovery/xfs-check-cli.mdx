À la console ou via %%SSH|ssh%%, exécutez :

```
xfs_repair -v /dev/mdXp1
```

Remplacez **X** par le bon numéro de disque (par exemple, `md1p1` pour Disque 1).

- Examinez le rapport de réparation pour des étapes de remédiation supplémentaires si nécessaire. Si seuls des problèmes mineurs sont présents, exécuter `-v` est généralement suffisant.
- Si la réparation produit un répertoire `lost+found`, examinez et gérez comme ci-dessus.
- %%Parity|parity%% est maintenu pendant la réparation.
- Une fois terminé, arrêtez le %%array|array%% et redémarrez en mode normal.
