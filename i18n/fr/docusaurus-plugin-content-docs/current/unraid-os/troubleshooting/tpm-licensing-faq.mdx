# FAQ de la licence TPM (7.3+)

Cette page contient des questions fréquemment posées sur la licence basée sur TPM dans Unraid OS. Pour les questions générales sur la licence, voir la [FAQ de la licence](/unraid-os/troubleshooting/licensing-faq/). Pour les questions sur le démarrage interne, voir la [FAQ du démarrage interne (7.3+)](/unraid-os/getting-started/set-up-unraid/internal-boot-faq/).

---

### Comment puis-je passer d'une licence basée sur flash à une licence basée sur TPM ? {/* #tpm */}

:::note[Unraid 7.3 et versions ultérieures]

La licence basée sur TPM est disponible dans Unraid 7.3 et les versions ultérieures.

:::

Toutes les nouvelles clés et les clés de remplacement utilisent une licence basée sur le TPM lorsque cela est possible, car elle est plus fiable qu’une licence basée sur le flash.

Pour passer proactivement à une licence basée sur TPM, accédez à ***Outils → Enregistrement*** et cliquez sur **Déplacer la licence vers TPM**. Vous êtes redirigé vers le site web Unraid Account, où vous pouvez approuver la demande de remplacement.

Après avoir cliqué sur **Déplacer la licence vers TPM**, le parcours Unraid Account affiche l'écran **Remplacer la clé de licence** :

![Écran Remplacer la clé de licence affiché après avoir cliqué sur Déplacer la licence vers TPM](/img/move-license-to-tpm-replace-key.png)

Si le bouton **Déplacer la licence vers TPM** n'est pas disponible, Unraid n'a pas pu détecter votre périphérique TPM 2.0. Il est peut-être désactivé dans votre BIOS. Vérifiez les paramètres liés à **Trusted Computing**, et laissez **Secure Boot** désactivé.

### Quels sont les avantages du passage à une licence basée sur TPM ? {/* #tpm-benefits */}

Les périphériques de démarrage USB traditionnels deviennent moins fiables pour certains utilisateurs. La licence basée sur TPM signifie que votre licence reste valide même si vous changez de matériel de démarrage.

### Que se passe-t-il si j'obtiens une nouvelle carte mère ? {/* #tpm-new-motherboard */}

Vous pouvez remplacer une licence basée sur TPM sur une nouvelle carte mère, de la même manière que vous remplacez une licence basée sur flash sur un nouveau périphérique USB. Dans ce cas, seule la licence d'origine est mise sur liste noire, et non la carte mère entière.

### Que se passe-t-il si je mets à jour mon BIOS ? Ai-je besoin d'une nouvelle licence ? {/* #tpm-bios */}

Le TPM ne devrait pas être affecté par les mises à jour du BIOS. Si, d'une manière ou d'une autre, c'est le cas, vous pouvez remplacer votre licence comme si vous aviez obtenu une [nouvelle carte mère](#tpm-new-motherboard).

### Que faire si ma carte mère n'a pas de périphérique TPM ? {/* #tpm-missing */}

Le passage à une licence basée sur TPM n'est pas une obligation absolue, mais de nombreuses cartes mères plus anciennes incluent un en-tête sur lequel un périphérique TPM additionnel peut être installé. Ces périphériques sont peu coûteux et largement disponibles.

### Puis-je passer à une licence basée sur TPM tout en démarrant depuis une clé flash ? {/* #tpm-flash-boot */}

Oui, la méthode de licence n'est pas liée à la méthode de démarrage.

Pour les questions spécifiques au démarrage interne, voir la [FAQ du démarrage interne (7.3+)](/unraid-os/getting-started/set-up-unraid/internal-boot-faq/).

### Puis-je utiliser TPM depuis une machine virtuelle Unraid ? {/* #tpm-vm */}

Non, ce n'est pas pris en charge.

### Puis-je déplacer ma licence entre TPM et USB si j'ai besoin de récupérer après une panne matérielle ? {/* #tpm-usb-replacement */}

Oui.

Le passage à une licence basée sur TPM met la licence d'origine sur liste noire sur le périphérique USB d'origine. À partir de là :

- Vous pouvez revenir vers un périphérique USB à l'aide du flux de remplacement de licence.
- Vous pouvez passer d'une carte mère équipée d'un TPM à une autre en utilisant le flux de remplacement TPM.

Si vous rencontrez une panne matérielle qui empêche le flux de remplacement normal, contactez le [support Unraid](https://support.unraid.net/support/home) pour obtenir de l'aide.

### Une réinitialisation du TPM affectera-t-elle ma licence Unraid ? {/* #tpm-reset */}

Non. La licence TPM d'Unraid utilise vos identifiants TPM, donc les réinitialisations du TPM ne devraient pas affecter votre licence Unraid.
