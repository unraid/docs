# Archives des versions

Utilisez cette archive pour trouver les notes de version et les liens de téléchargement des versions d’Unraid OS. Pour mettre à jour un serveur existant, utilisez le programme de mise à jour intégré ; consultez le [processus de mise à jour standard](/unraid-os/system-administration/maintain-and-update/upgrading-unraid#standard-update-process) pour connaître les étapes. Pour créer un nouveau périphérique de démarrage, utilisez le [créateur USB Unraid](https://unraid.net/download).

L’archive regroupe les versions par statut de support. Seule la version de correctif la plus récente ou la version préliminaire la plus récente d’une série mineure prise en charge reçoit les mises à jour, selon le cas ; les versions plus anciennes de la même série sont remplacées. Les versions marquées **EOL** ne sont pas prises en charge et ne reçoivent plus de mises à jour.

Si vous devez choisir quelle version installer, commencez par [types de versions d’Unraid OS](./updating-unraid/release-types.mdx). Pour obtenir des détails sur l’éligibilité de la licence, consultez la [FAQ sur les licences](./troubleshooting/licensing-faq.mdx#no-extension).
