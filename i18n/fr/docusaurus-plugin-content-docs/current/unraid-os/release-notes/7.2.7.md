# Notes de version · Unraid OS 7.2.7

## Version 7.2.7 2026-05-21

Cette version met l'accent sur les mises à jour de sécurité, les mises à jour de Docker et les corrections de WebGUI, ainsi que sur le rafraîchissement des paquets des composants principaux du système.

Pour les environnements sensibles à la sécurité, la mise à niveau est recommandée.

## Mise à niveau

Pour des instructions étape par étape, consultez [Mise à jour d'Unraid](/unraid-os/system-administration/maintain-and-update/upgrading-unraid/). Des questions sur votre [licence](/unraid-os/troubleshooting/licensing-faq/) ?

## Modifications par rapport à la [version 7.2.6](/unraid-os/release-notes/7.2.6/)

### Sécurité

- Mises à jour de sécurité pour docker, nginx, glibc, jq, avahi et les bibliothèques de support de glibc.

### Conteneurs / Docker

- Docker 29.5.1
- Les conteneurs Docker VLAN conservent leur passerelle configurée après les mises à jour.
- Les menus des conteneurs Docker restent accessibles sur les pages comportant de nombreux conteneurs.

### WebGUI / Système

- Les vérifications de version Connect sont rétablies dans le module serveur.

### Noyau Linux

- mise à niveau du noyau : 6.12.90-Unraid (CVE-2026-43490, CVE-2026-46333)
- Régression de performances affectant les systèmes avec Docker activé et certains périphériques USB 3.1.
