# Version 7.2.2 2025-11-21

Si vous êtes déjà sur la version 7.2.1, cette mise à jour est **facultative**. Elle a été principalement publiée pour les utilisateurs en version d'essai, pour corriger un bug qui les empêchait de voir la date d'expiration de leur essai. Nous avons aussi résolu un problème lié aux couleurs d'arrière-plan personnalisées de l'en-tête, nous recommandons de mettre à jour si vous avez ces paramètres.

Si vous êtes sur la version 7.2.0 ou antérieure, vous pouvez passer directement à la 7.2.2.

## Mise à niveau

Pour des instructions étape par étape, voir [Mise à jour de Unraid](../../unraid-os/system-administration/maintain-and-update/upgrading-unraid.mdx).

### Problèmes connus

Pour d'autres problèmes connus, voir les [notes de version 7.2.1](7.2.1.md#known-issues).

### Rétrogradation

Si vous revenez en arrière plus tôt que 7.2.1, consultez aussi les [notes de version 7.2.1](7.2.1.md#rolling-back).
