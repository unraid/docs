<h4>Méthode terminal (recommandée)</h4>

1. Installez `wakeonlan` en utilisant Homebrew :

   ```
   brew install wakeonlan
   ```

2. Envoyez le paquet magique :

   ```
   wakeonlan MAC_ADDRESS
   ```

<h4>Alternative GUI : miniWOL</h4>

1. Téléchargez [miniWOL v3](https://apps.apple.com/app/miniwol/id6474282023) depuis l'App Store.
2. Ajoutez votre serveur en utilisant son adresse MAC.
3. Cliquez sur l'icône du soleil pour réveiller votre serveur.
