:::note[WebGUI: Périphérique de démarrage contre Flash]

Dans Unraid 7.3.0 et les versions ultérieures, le %%WebGUI|web-gui%% utilise **Périphérique de démarrage** dans l’onglet **Main** (par exemple les paramètres et les sauvegardes **Boot Device**). Dans les versions antérieures, les mêmes contrôles sont libellés **Flash** ou **Flash device**. Utilisez l’intitulé affiché par votre version.

:::
